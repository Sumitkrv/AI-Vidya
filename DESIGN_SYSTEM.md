## 5ï¸âƒ£ ANIMATION PLAN

### **SCROLL-TRIGGERED ANIMATIONS**

```javascript
// Using Framer Motion or GSAP

1. Fade Up + Blur
   - Element starts 50px below, blurred
   - Fades in + moves up as scroll reaches
   - Used: Headings, cards, images

2. Stagger Children
   - Parent container triggers
   - Children animate in sequence with delay
   - Used: Category grid, course grid

3. Number Counter
   - Counts from 0 to target number
   - Triggers when in viewport
   - Used: Stats section

4. Draw Line
   - SVG path draws progressively
   - Used: How It Works connecting line

5. Scale In
   - Starts at scale(0.8), opacity 0
   - Animates to scale(1), opacity 1
   - Used: Feature cards

6. Parallax Layers
   - Background moves slower than foreground
   - Creates depth illusion
   - Used: Hero section, decorative orbs
```

### **HOVER ANIMATIONS**

```javascript
1. Magnetic Effect
   - Element follows cursor within boundary
   - Spring physics (damping: 15, stiffness: 150)
   - Used: Primary buttons, course cards

2. Glow Effect
   - Box shadow expands + brightens
   - Duration: 300ms ease-out
   - Used: All cards, buttons

3. 3D Tilt
   - Rotates based on cursor position
   - Subtle (max 10 degrees)
   - Used: Course cards, testimonials

4. Lift Effect
   - translateY(-8px) + shadow increase
   - Used: Category cards

5. Shine Sweep
   - Linear gradient sweeps across
   - Duration: 600ms
   - Used: CTA buttons

6. Icon Bounce
   - Scale + rotate animation
   - Used: Feature icons on hover
```

### **PAGE LOAD ANIMATIONS**

```javascript
1. Hero Entrance
   - Logo fades in
   - Headline splits + slides up
   - Buttons fade in with delay
   - Image slides from right with blur

2. Floating Orbs
   - Continuous slow movement
   - Random paths
   - Blur pulse effect

3. Gradient Shift
   - Background gradient slowly shifts hue
   - Subtle, infinite loop
```

### **MICRO-INTERACTIONS**

```javascript
1. Button Click
   - Scale down to 0.95
   - Ripple effect from center
   - Returns to 1 with bounce

2. Input Focus
   - Border color transition
   - Label slides up + shrinks
   - Glow appears

3. Checkbox/Toggle
   - Smooth slide animation
   - Color transition
   - Haptic-like feedback

4. Cursor Ripple
   - Custom cursor with glow trail
   - Enlarges on interactive elements
```

## 9ï¸âƒ£ FINAL POLISHED UI DESCRIPTION

### **OVERALL AESTHETIC**

The platform should feel like **"Apple meets Stripe meets Awwwards"** â€” a perfect balance of:
- **Minimalism** (lots of breathing space)
- **Premium Details** (micro-animations, glow effects)
- **Dark Elegance** (deep blacks with neon accents)
- **Futuristic Tech** (3D elements, glass morphism)

### **VISUAL HIERARCHY**

1. **Headers**: Massive, bold, gradient text
2. **Body**: Clean, readable, soft white on dark
3. **CTAs**: Vibrant gradients, impossible to miss
4. **Images**: Sharp, consistent purple/blue lighting
5. **Cards**: Glass borders, subtle shadows, hover glow

### **SPACING & RHYTHM**

- **Section Padding**: 120px top/bottom (desktop)
- **Grid Gaps**: 32px between cards
- **Max Width**: 1400px container
- **Border Radius**: 24px (2xl) for cards
- **Rounded Buttons**: 16px (xl)

### **DEPTH & LAYERS**

1. **Background Layer**: Deep black with subtle gradient
2. **Decorative Layer**: Floating gradient orbs, grid patterns
3. **Content Layer**: Cards, images, text
4. **Interactive Layer**: Buttons, links with hover states
5. **Overlay Layer**: Glassmorphism elements, tooltips

### **MOTION PHILOSOPHY**

- **Fast**: 200ms (micro-interactions, button hovers)
- **Medium**: 400ms (card animations, reveals)
- **Slow**: 800ms (page transitions, parallax)
- **Easing**: Custom cubic-bezier(0.4, 0, 0.2, 1) for smoothness

### **ACCESSIBILITY**

- **Contrast Ratio**: Minimum 4.5:1 (WCAG AA)
- **Focus States**: Visible purple outline (2px)
- **Font Size**: Minimum 16px body text
- **Touch Targets**: Minimum 44x44px
- **Keyboard Navigation**: Full support
- **Screen Readers**: Semantic HTML, ARIA labels

### **RESPONSIVE BEHAVIOR**

- **Desktop (1400px+)**: Full layout, all animations
- **Laptop (1024-1399px)**: Slightly reduced spacing
- **Tablet (768-1023px)**: 2-column grid, simplified parallax
- **Mobile (<768px)**: Single column, bottom nav, simplified animations
